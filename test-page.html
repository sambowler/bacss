BaCSS is a stylesheet that you can plug in to any site that is under construction. It will add basic styling, spacing and layout to every possible element you could imagine.

There are two ways of using BaCSS.

The traditional method

If you're using traditional ID's for your layout such as #header, #main, #sidebar, #footer or you're using HTML5's new semantic elements, this is probably the solution for you. SImply include the stylesheet as you would any other. 

                <link rel="stylesheet" href="bacss-0.1.css" media="screen" type="text/css">

The other method

You can also customise BaCSS to use any custom ID's you would like. You can do this by either simply editing the stylesheet or using the JavaScript solution.
