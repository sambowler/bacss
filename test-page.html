Huh, what?!

BaCSS (pronounced “base”) is a stylesheet that is just that, a base set of styling for anything you’re building. It will add basic styling to anything and everything. “Why?” You may ask. Because nobody wants to show clients default browser styling. This page is using it.

Download it here.

Jus' like 'at

If you need IE support

You can include the JavaScript version of BaCSS. This will pull in the latest CSS as well as ie7-js which is required for CSS3 selector support.

<script src="http://github.com/sambowler/bacss/raw/master/latest.js"></script>

If IE support is no biggy

Just include the CSS or download to customise to your needs (you can drop me an email if there's something that you feel should be included and other people would benefit from).

<link rel="stylesheet" href="http://github.com/sambowler/bacss/raw/master/latest.css" media="screen">
